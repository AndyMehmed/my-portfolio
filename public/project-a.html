🚀 Project A

Ett spännande projekt där jag lärde mig nya tekniker och färdigheter

🔍 Vad handlar projektet om?

Project A var en utmaning där jag fick arbeta med frontend-utveckling och lärde mig att hantera komponentbaserad utveckling i React. Jag använde även olika API:er för att hämta data dynamiskt.

📌 Teknologier jag använde

  • ✅ React
  • ✅ CSS för styling
  • ✅ API-hantering med Fetch/Axios
  • ✅ Git & GitHub för versionshantering

💡 Vad jag lärde mig

Jag fick en djupare förståelse för komponentbaserad utveckling, tillståndshantering och responsiv design. Jag lärde mig också att strukturera en kodbas på ett bättre sätt för att göra den mer skalbar och lättläst.
